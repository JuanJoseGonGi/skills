# Software Engineering Skills

A collection of skills and best practices for modern software development.

## Credits

Some of the guides in this repository (such as the Go development guide) were adapted and translated from the excellent work of [sumik5](https://github.com/sumik5/dotfiles). Additionally, the CodeGuard security rules and skill are provided by [Project CodeGuard](https://github.com/project-codeguard/rules). We thank the authors for providing such high-quality resources.

## Contents

- **Developing Go**: Comprehensive guide for Go development following Google and community best practices.
- **Applying SOLID Principles**: Practical application of clean code and SOLID design patterns.
- **Enforcing Type Safety**: Principles for robust type-safe implementations in TypeScript and Python.
- **Authoring Skills**: Guide for creating effective AI Agent Skills following the official specification.
- **Testing**: A test-first approach focusing on TDD, the Test Pyramid, and testable design.
- **Securing Code**: Implementation of OWASP countermeasures and security best practices.
- **CodeGuard Security**: Integrated security rules and checklists from Project CodeGuard.
- **Writing Dockerfiles**: Optimization and security for containerized environments.
