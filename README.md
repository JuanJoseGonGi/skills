# Software Engineering Skills

A collection of skills and best practices for modern software development.

## Credits

Some of the guides in this repository (such as the Go development guide) were adapted and translated from the excellent work of [sumik5](https://github.com/sumik5/dotfiles). We thank the original author for providing such high-quality resources.

## Contents

- **Developing Go**: Comprehensive guide for Go development following Google and community best practices.
- **Applying SOLID Principles**: Practical application of clean code and SOLID design patterns.
- **Testing**: A test-first approach focusing on TDD, the Test Pyramid, and testable design.
- **Securing Code**: Implementation of OWASP countermeasures and security best practices.
- **Writing Dockerfiles**: Optimization and security for containerized environments.
